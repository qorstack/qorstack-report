# Use Yarn

Always use **Yarn** as the package manager. Do not use npm.

## Rules

- Use `yarn add` instead of `npm install`
- Use `yarn remove` instead of `npm uninstall`
- Use `yarn dev`, `yarn build`, `yarn start` instead of `npm run dev`, `npm run build`, `npm run start`
- Never generate `package-lock.json` — only `yarn.lock` is used
